Teknisk opsætning

Nedenfor er en liste over konfigurable indstillinger, som opsættes af driftoperatøren. Hvis disse indstillinger skal ændres, skal der tages kontakt til driftoperatøren via helpdesk@digital-identity.dk. Bemærk at flere af disse indstillinger kan kræve aftaleindgåelse hos 3.part, fx Digitaliseringsstyrelsen eller KOMBIT.
